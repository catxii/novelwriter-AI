# 📋 网文写作清单

> 对照使用，每次开写前和写完一章后各检查一遍。

---

## 🔲 一、开写前必做

- [ ] 用一句话说清你的书（主角是谁 + 要什么 + 阻碍 + 核心卖点）
- [ ] 确定题材和基调
- [ ] 写主角卡：目标 / 优势 / 弱点 / 当前困境
- [ ] 定至少3个主要人物的动机
- [ ] 写简版大纲：开篇 → 几个关键转折 → 第一个大高潮 → 结尾方向
- [ ] 想清楚你的"爽点/情感点/悬念点"分别是什么

---

## 🔲 二、黄金三章检查

- [ ] 主角第1章就出场了吗？
- [ ] 第1章有冲突或事件发生吗？
- [ ] 前三章展示了这本书的核心卖点了吗？
- [ ] 每章结尾有钩子（悬念 / 反转 / 期待感）吗？
- [ ] 没有大段背景/设定说明（流量杀手）？
- [ ] 没有低级错误（人名混乱 / 逻辑漏洞）？

---

## 🔲 三、每章写完自查

- [ ] 这章推进了什么？（剧情 / 关系 / 信息 / 情绪）
- [ ] 这章删掉会影响故事吗？（不影响就重写）
- [ ] 有情绪波动吗？（不要全程平铺）
- [ ] 有没有制造下一章的期待？
- [ ] 对话像"人话"吗？（不生硬 / 不像独白）

---

## 🔲 四、人物检查

- [ ] 主角有欲望、会行动、会改变局面吗？
- [ ] 主角不只是"被剧情推着走"？
- [ ] 每个配角都有存在功能（制造冲突 / 提供帮助 / 形成对照）？
- [ ] 反派有动机，不只是"坏"？

---

## 🔲 五、节奏检查

- [ ] 前三章没有超过5000字还没进主线？
- [ ] 没有一个事件拖超过10章？
- [ ] 小高潮频率够吗？（每5~10章至少有一次）
- [ ] 大高潮之前蓄力了足够久吗？

---

## 🔲 六、设定检查

- [ ] 世界观是通过剧情"呈现"出来的，不是直接"讲解"的？
- [ ] 设定为冲突服务，不是为炫耀服务？
- [ ] 核心规则清晰，读者不会懵？

---

## 🔲 七、连载意识

- [ ] 更新节奏稳定（宁可少更，不要断更）？
- [ ] 每章字数大致统一（建议 2000~4000 字/章）？
- [ ] 有没有存稿缓冲（至少3~5章）？
- [ ] 推荐期间更新速度有没有提速？

---

## 🔲 八、随时问自己这一句

> **如果我是读者，我为什么要点下一章？**

答得出来 → 继续写  
答不出来 → 先想清楚再写

---

## ⚡ 口诀

> 开头要快，主角要强，目标要清，冲突要早。  
> 每章有事，每段有钩，设定服务剧情，人物服务情绪。  
> 小高潮不断，大高潮蓄力，连载求稳定，长期靠节奏。
